# 페이지 포함기술
* 페이지 공통부를 클라이언트가 요청시 요청페이지와 합쳐서 응답하는 기술.
* 페이지 내용을 포함하는 3가지 방식
  - directive(지시자) - include
  - action(액션) - include
  - jstl - import

<br><br><br><br>

# directive

```java

<%@ include file="common.jsp" %> 

```
