# EL표현식이란?
* jsp파일에 다양한 언어가(html, javascript, java 등) 섞이면서 코드해석이 불편해졌다.


```java

$ { expr }

```

* 형식으로 사용하고, expr은 자바의 변수가 아니다!






# EL에서 사용하는 연산자
* 키워드 연산자를 지원한다.
|연산자|종류
|:---:|---|
|산술연산자|ㅁ|
|비교연산자|ㅁ|
|논리연산자|ㅇ|
|empty 연산자|ㅇ|
|삼항연산자|ㅇ|
