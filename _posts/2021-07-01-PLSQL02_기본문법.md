# 기본문법
## 1. Block

```sql
DECLARE
  V_EMPNO NUMBER(4)   :=0;
  V_ENAME VARCHAR2(10);
  V_DEPTNO NUMBER(2);
BEGIN
  V_EMPNO := 7778;
  V_ENAME := 'PL/SQL';
  INSERT INTO EMP(EMPNO, DEPTNO, ENAME) VALUES(V_EMPNO, V_DEPTNO, V_ENAME);
  COMMIT;
EXCEPTION
  WHEN OTHERS THEN
    DBMS_OUTPUT.PUT_LINE('INSERT ERR : '||SQLERRM);
    ROLLBACK;
END;
/
```

* SQLERRM : 오류메시지 반환
* WHEN 예외명 THEN 예외처리구문<br>...<br>WHEN OTHERS THEN 예외처리 구문 : 구체적인 예외명을 알 수 없을 때 사용.
