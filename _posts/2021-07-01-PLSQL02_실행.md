# PL/SQL
* 별도의 디버깅 도구를 지원하지 않음.
* DBMS_OUTPUT_1.SQL

```sql
REM DEFAULT SIZE 20000 BYTES.
SET SERVEROUTPUT ON
BEGIN
  FOR I IN 1..10
  LOOP
    DBMS_OUTPUT.PUT_LINE('[ '||TO_CHAR(I)||' ] PROCESSED');
END;
/
  SET SERVEROUTPUT OFF
  BEGIN
    DBMS_OUTPUT.PUT_LINE('OK...');
END;
/
```

* DBMS_OUTPUT_2.SQL
